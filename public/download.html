Desktop Features

  • トレイ常駐と hide-on-close
  • グローバルショートカット: Windows は Ctrl + Shift + Space、macOS は Command + Shift + Space
  • 通常画面と同じタイム、計算、議事録、マイクが使えるオーバーレイ
  • オーバーレイの自動保存、新規メモ作成、透明度調整、ドラッグ移動
  • 自動起動、ファイルエクスポート、自動更新

Known Notes

  • macOSでは完全なネイティブ透明ウィンドウではなく、React側の透明度でオーバーレイを表現します。
  • Intel Mac向けのビルドは次回以降の対応予定です。
  • デスクトップ版の直接リンクは、GitHub Release作成後に有効になります。
  • マイクやショートカットはOS側の権限許可が必要になる場合があります。
  • Android APKでは提供元不明アプリの確認画面が表示される場合があります。

Browser Install

Chrome / Edge では、ブラウザ版をホーム画面やデスクトップに追加できます。

お使いのブラウザはインストールに対応していません。
