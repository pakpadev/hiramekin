# Hiramekin v1 ビジュアルリデザイン仕様

**日付**: 2026-05-22
**ステータス**: 承認済み

## 概要

v1 リリース後のビジュアル改善。モノクロベース＋シアンアクセント＋シマーボーダーアニメーションの統一テーマへ移行し、ダウンロードページを一般ユーザーでも迷わない構成に刷新する。

---

## 1. アプリ テーマ

### カラーパレット

| 役割 | 値 | 用途 |
|---|---|---|
| 背景 | `#111111` | アプリ全体背景 |
| サーフェス | `#1a1a1a` | カード・入力欄背景 |
| テキスト主 | `#ffffff` | ヘッダー・ラベル |
| テキスト本文 | `#cccccc` | メモ本文 |
| テキストミュート | `#444444` | プレースホルダー・セクションラベル |
| アクセント | `#00e5ff` | FAB・アクションボタン・ピン・リンク・CalcLine ボーダー |
| 危険 | `#ff4c4c` | アーカイブボタン |

### シマーボーダーアニメーション

入力欄とメモカードの枠線に適用する。

```
グラデーション: #111111 → #00e5ff → #b2f0ff → #111111
方向: 左から右
background-size: 300% 100%
duration: 2.4s linear infinite
```

カード間の単調さを避けるため、各カードに `animation-delay` をインデックスベースでずらす（例: index * -0.6s）。

実装パターン（React Native では `LinearGradient` + `Animated` を使用）:

```tsx
// Web (CSS)
<div style={{
  padding: 1.5,
  borderRadius: 10,
  background: 'linear-gradient(90deg, #111, #00e5ff, #b2f0ff, #111)',
  backgroundSize: '300% 100%',
  animation: 'shimmer 2.4s linear infinite',
}} />
```

React Native（Expo）では `expo-linear-gradient` + `Animated.loop` を使用する。`border` プロパティでは gradient が使えないため、ラッパー View に gradient を背景として敷き、内側 View で背景色を塗って枠線のように見せる手法を採る。

### ダークモード

既存のライトモード分岐（`useColorScheme`）は削除し、常時ダークテーマに統一する。理由: ライトモードではシアンのコントラストが不十分で、シマーアニメーションの視認性が落ちる。

### 対応コンポーネント

| ファイル | 変更内容 |
|---|---|
| `App.tsx` | `theme` オブジェクトをシアン＋ダークに固定。FAB 色を `#00e5ff`（文字色 `#000`）に変更。FAB に `shadowColor: '#00e5ff', shadowOpacity: 0.5, shadowRadius: 10, elevation: 8` |
| `src/components/InputArea.tsx` | ラッパー View にシマーグラデーション適用 |
| `src/components/MemoItem.tsx` | カードラッパーにシマーグラデーション適用。`animationDelay` を index prop で受け取る |
| `src/components/CalcLine.tsx` | 左ボーダー色を `#00e5ff`、背景を `rgba(0,229,255,0.07)` に変更 |
| `src/components/KeyboardToolbar.tsx` | アクティブチップ: 背景 `rgba(0,229,255,0.12)`、ボーダー `rgba(0,229,255,0.3)`、テキスト `#00e5ff` |
| `src/components/SearchBar.tsx` | ボーダー色を `#2a2a2a`、背景を `#1a1a1a` に変更 |

## 2. ダウンロードページ

### 構成（ビッグボタン＋FAQ 型）

```
ヒーローセクション
  アイコン（88px、角丸）
  アプリ名「Hiramekin」
  バージョンタグ「v1.0.1 beta · Android」
  シマーボーダー付きダウンロードボタン「無料でダウンロード」
  サブテキスト「56 MB · Android 6.0 以上」

FAQ セクション（アコーディオンなし、常時展開）
  Q1. 「提供元不明」と出た — 大丈夫?
  Q2. インストールできない
  Q3. Play Store にないの?

フッター（最小表示）
  バージョン / ビルド情報（小テキスト）
```

### 言語・表記の変更

| 削除・変更前 | 変更後 |
|---|---|
| APK / APKをダウンロード | アプリ / 無料でダウンロード |
| Android sideload | （削除） |
| SHA-256 ハッシュ | （削除） |
| 提供元不明のアプリとして警告が表示される場合があります | FAQ Q1 に移動し「安全です」と明記 |
| GitHub Issues に記録してください | （削除） |
| インストール後、計算、音声入力などを確認してください | （削除 — ベータテスター向け文言のため） |

### FAQ テキスト（確定）

**Q: 「提供元不明」と出た — 大丈夫?**
> Play Store 以外からインストールするアプリには必ず出る確認画面です。「この提供元を許可」または「許可」を選ぶと進められます。

**Q: インストールできない**
> 設定 → セキュリティ →「不明なアプリのインストール」でお使いのブラウザを許可してください。

**Q: Play Store にないの?**
> 現在ベータ版のため直接配布しています。正式版は Play Store での公開を予定しています。

### テーマ

`download.html` もアプリ本体と同じカラーパレット（`#111` 背景、`#00e5ff` アクセント）を適用し、ダウンロードボタンにシマーアニメーションを付ける。
