# v1.0.8-beta

## Changes

- Fixed stale Service Worker navigation caching that could leave returning web users on a blank screen.
- Changed archive from a always-visible memo action to a left-swipe action with an explicit archive affordance.
- Added an archive undo toast: `アーカイブしました / 元に戻す`.
- Increased memo action and toolbar tap targets for mobile use.
- Improved placeholder, button, icon, and surface contrast in both light and dark mode.
- Updated Jest setup to mock Expo vector icons in component tests.

## Verification

- `npm run typecheck`
- `npm test -- --runInBand`
- `npm run build:web`
- Playwright screenshots for mobile light, mobile dark, and desktop light.
