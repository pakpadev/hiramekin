# Hiramekin v1.0.1 beta

## 概要

Hiramekin v1 の Android 実機検証用 APK リリース。

このリリースは Google Play 未公開の直接配布版です。v1.0.0 beta から公開配布前のセキュリティ調整を反映しています。

## APK

- ファイル名: `hiramekin-v1.0.1-beta.apk`
- 配布URL: `https://github.com/pakpadev/hiramekin/releases/download/v1.0.1-beta/hiramekin-v1.0.1-beta.apk`
- サイズ: 58,647,852 bytes
- 表示サイズ: 56 MB
- SHA-256: `EB00A7E277CFEE1A2D4A9A42579D92E62197B23CEB57E61E4EC79213777D2DE4`

## v1.0.0 beta からの変更

- Android `versionCode` を 2 に更新。
- Android `versionName` を `1.0.1-beta` に更新。
- 計算式評価を四則演算文字だけに制限。
- Android backup を無効化。
- 不要な外部ストレージ権限とランチャーバッジ権限を削除。
- dev client network inspector を無効化。

## 動作確認項目

- アプリが起動し、メモ入力欄にすぐ入力できる。
- メモを保存できる。
- 検索で対象メモを絞り込める。
- ピンとアーカイブが動作する。
- `100+200` と `=100+200` の計算結果が表示される。
- タイム、計算、議事録のクイック挿入が動作する。
- マイク権限が表示され、音声入力した文字が挿入される。
- 通知権限が表示され、スケジュール通知が届く。
- ダークモードで文字が読める。

## 既知の制約

- Android 専用。iOS は対象外。
- debug signing config 署名のため、Play Store 提出用ではない。
- 自動更新は未対応。
- Google Play Protect やブラウザが提供元不明アプリの警告を出す場合がある。

## インストール手順

1. Android 端末で `download.html` を開く。
2. `APKをダウンロード` を押す。
3. ダウンロードしたAPKを開く。
4. 警告が出た場合は、ブラウザからのインストールを一時的に許可する。
5. インストール後、上記の動作確認項目を確認する。
