# Hiramekin v1.0.5 beta

## 概要

Hiramekin v1 の Android 実機検証用 APK リリース。

このリリースは Google Play 未公開の直接配布版です。v1.0.4 beta からコピーボタンの視認性改善を反映しています。

## APK

- ファイル名: `hiramekin-v1.0.5-beta.apk`
- 配布URL: `https://github.com/pakpadev/hiramekin/releases/download/v1.0.5-beta/hiramekin-v1.0.5-beta.apk`
- サイズ: 58,676,740 bytes
- 表示サイズ: 56 MB
- SHA-256: `AF1EE87FA73D58272F4728310B44F945E175D1FC843B0C80FAEEB6EBA8C35235`

## v1.0.4 beta からの変更

- メモカードのコピーボタンアイコンを `⎘` から `⧉` に変更。
- コピーボタンを常時シアン色で表示し、視認性を向上。

## 動作確認項目

- アプリが起動し、メモ入力欄にすぐ入力できる。
- メモを保存できる。
- 検索で対象メモを絞り込める。
- ピンとアーカイブが動作する。
- `100+200` と `=100+200` の計算結果がシアン色で表示される。
- タイム、計算、議事録のクイック挿入が動作する。
- マイク権限が表示され、音声入力した文字が挿入される。
- 通知権限が表示され、スケジュール通知が届く。
- ダークモードとライトモードで文字が読める。
- メモカードに ⧉ (シアン) が表示され、タップでコピー、✓ に変わる。

## 既知の制約

- Android 専用。iOS は対象外。
- debug signing config 署名のため、Play Store 提出用ではない。
- 自動更新は未対応。
- Google Play Protect やブラウザが提供元不明アプリの警告を出す場合がある。
