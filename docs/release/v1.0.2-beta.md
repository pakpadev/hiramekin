# Hiramekin v1.0.2 beta

## 概要

Hiramekin v1 の Android 実機検証用 APK リリース。

このリリースは Google Play 未公開の直接配布版です。v1.0.1 beta からビジュアルリデザインを反映しています。

## APK

- ファイル名: `hiramekin-v1.0.2-beta.apk`
- 配布URL: `https://github.com/pakpadev/hiramekin/releases/download/v1.0.2-beta/hiramekin-v1.0.2-beta.apk`
- サイズ: 58,652,468 bytes
- 表示サイズ: 56 MB
- SHA-256: `0311ED5DFCEF8E89A065D27EA6F52FEE4A311B621729DA4F4D8EAB5F161E372D`

## v1.0.1 beta からの変更

- モノクロ＋シアン (`#00e5ff`) のカラーテーマへ全面移行。
- 入力欄とメモカードの枠線にシマーボーダーアニメーションを追加。
- メモ一覧をカードスタイルに変更（フラットリストから）。
- ライトモード対応: 端末設定に追従し、ライト時はシアン `#00acc1` を使用。
- ダウンロードページを一般ユーザー向けに刷新（ビッグボタン＋FAQ形式）。
- 技術用語（APK・sideload・SHA-256・GitHub Issues）をページ本文から削除。

## 動作確認項目

- アプリが起動し、メモ入力欄にすぐ入力できる。
- メモを保存できる。
- 検索で対象メモを絞り込める。
- ピンとアーカイブが動作する。
- `100+200` と `=100+200` の計算結果がシアン色で表示される。
- タイム、計算、議事録のクイック挿入が動作する。
- マイク権限が表示され、音声入力した文字が挿入される。
- 通知権限が表示され、スケジュール通知が届く。
- ダークモードとライトモードで文字が読める。
- シマーボーダーアニメーションが入力欄とカードで動作している。

## 既知の制約

- Android 専用。iOS は対象外。
- debug signing config 署名のため、Play Store 提出用ではない。
- 自動更新は未対応。
- Google Play Protect やブラウザが提供元不明アプリの警告を出す場合がある。
