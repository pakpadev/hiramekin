# v1.0.9 Desktop Release

## 概要

Hiramekin v1.0.9 は、既存の高速メモ体験を Windows / macOS のデスクトップアプリとして使えるようにしたリリースです。

## 追加機能

- デスクトップアプリ化
  - Tauri v2 による Windows / macOS 向けビルドに対応。
- システムトレイ
  - 閉じても常駐し、トレイから表示・終了できる。
- グローバルショートカット
  - Windows / Linux: `Ctrl + Shift + Space`
  - macOS: `Command + Shift + Space`
  - ショートカットでオーバーレイを表示・非表示できる。
- オーバーレイ
  - 通常画面と同じタイム、計算、議事録、マイクを利用可能。
  - 保存ボタンなしで自動保存。
  - 新規メモ作成に対応。
  - 通常時の透明度を5%刻みで調整可能。
  - ホバー時は75%固定で視認性を上げる。
  - ヘッダー中央の空白領域をドラッグして移動可能。
- 自動起動
  - 設定画面からOS起動時の自動起動を切り替え可能。
- ファイルエクスポート
  - メモをMarkdown / JSON / CSVで保存可能。
- 自動更新
  - Tauri updater に対応。
  - GitHub Releases の署名付き成果物と `public/latest.json` で配信する。
- アプリアイコン
  - 「閃」一文字のデスクトップアプリアイコンを追加。

## 配布方針

- ファイル保管先は GitHub Releases。
- ユーザーの配布導線はダウンロードページ `public/download.html` に集約する。
- ダウンロードページからOS別のインストーラーを直接ダウンロードさせる。
- Windows は `.exe` / `.msi` 形式のインストーラーを配布する。
- macOS は `.dmg` 形式のインストーラーを配布する。
- 自動更新用に、Actions成果物に含まれる `.app.tar.gz` と `.sig` もRelease assetsとして保持する。
- Android beta APK は既存の `v1.0.9-beta` Release を継続利用する。

## 想定Release

- Tag: `desktop-v1.0.9`
- Release title: `Hiramekin v1.0.9 Desktop`
- Windows asset例: `Hiramekin_1.0.9_x64-setup.exe`
- Windows asset例: `Hiramekin_1.0.9_x64_en-US.msi`
- macOS asset例: `Hiramekin_1.0.9_x64.dmg`
- macOS updater asset例: `Hiramekin.app.tar.gz`

実際のasset名は GitHub Actions の `Desktop Build` Artifacts から取得したファイル名に合わせる。
ただし、ダウンロードページの直リンクを安定させるため、Releaseへアップロードするasset名は上記の名前に揃える。

## 自動アップデート

`public/latest.json` は、Release作成後に署名付き成果物のURLと署名を反映する。

現時点では誤ったURLや署名でアップデーターを壊さないよう、`platforms` は空のまま維持する。

Release後に必要な更新:

```json
{
  "version": "1.0.10",
  "notes": "更新内容",
  "pub_date": "2026-05-31T00:00:00Z",
  "platforms": {
    "windows-x86_64": {
      "signature": "<Hiramekin_1.0.10_x64-setup.exe.sig の内容>",
      "url": "https://github.com/pakpadev/hiramekin/releases/download/desktop-v1.0.10/Hiramekin_1.0.10_x64-setup.exe"
    },
    "darwin-x86_64": {
      "signature": "<Hiramekin_1.0.10_x64.dmg.sig の内容>",
      "url": "https://github.com/pakpadev/hiramekin/releases/download/desktop-v1.0.10/Hiramekin_1.0.10_x64.dmg"
    }
  }
}
```

同一バージョン `1.0.9` のアプリに `1.0.9` を配信しても更新通知は出ないため、自動アップデート検証は次バージョン `1.0.10` 以降で行う。

## 既知の制限

- macOSではビルド安定性を優先し、ネイティブウィンドウの完全透明化は使っていない。
  - オーバーレイの半透明表現はReact側のUI透明度で行う。
- macOSのグローバルショートカットやマイクは、OS側の権限設定が必要になる場合がある。
- 自動アップデートは、GitHub Release asset URL と `.sig` の反映後に有効になる。
